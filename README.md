# Space
VSCode Project Manager for Windows made in Electron.

> [!IMPORTANT]
> The only supported operating system is Windows.

> [!WARNING]
> Although it may run on Linux, full functionality of the program is not guaranteed.

## Installation
```
npm install
```

## Start
```
npm run start
```

## Build
```
npm run build
```
